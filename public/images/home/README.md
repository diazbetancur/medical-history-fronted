# Home Page Images

Imágenes para la página principal (Home).

## Imágenes Requeridas

### Hero Section
- **hero-bg.jpg / hero-bg.webp**
  - Dimensiones: 1920x1080px (Full HD)
  - Formato: WebP (preferido) o JPEG
  - Tamaño máximo: 200KB
  - Descripción: Imagen de fondo del hero section
  - Tema: Médicos, hospital, salud

### Features Section
- **feature-verified.jpg** (opcional)
  - Dimensiones: 800x600px
  - Ilustración de verificación/certificación

- **feature-schedule.jpg** (opcional)
  - Dimensiones: 800x600px
  - Ilustración de calendario/agenda

- **feature-quality.jpg** (opcional)
  - Dimensiones: 800x600px
  - Ilustración de calidad médica

## Uso en Código

```typescript
// En home.page.scss
.hero {
  background-image: url('/images/home/hero-bg.webp');
}
```
