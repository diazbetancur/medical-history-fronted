# Medical Specialties Images

Iconos e imágenes para especialidades médicas.

## Especificaciones

### Specialty Icons
- **Dimensiones**: 128x128px o 256x256px
- **Formato**: SVG (preferido) o PNG con transparencia
- **Estilo**: Line icons o filled icons consistentes
- **Color**: Preferiblemente monocromático (se colorea con CSS)

### Specialty Badges
- **Dimensiones**: 400x400px
- **Formato**: PNG con transparencia o SVG
- **Uso**: Badges grandes para páginas de especialidad

## Nomenclatura

```
[specialty-slug].svg              # Icono principal
[specialty-slug]-badge.svg        # Badge grande
[specialty-slug]-illustration.svg # Ilustración decorativa
```

## Uso en Código

```html
<!-- En specialty-chip o cards -->
<mat-icon>
  <img src="/images/specialties/cardiology.svg" alt="Cardiología">
</mat-icon>
```
